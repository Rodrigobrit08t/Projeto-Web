Nome do Projeto: Gerenciador de Tarefas com Integrações API

Descrição:
Este projeto consiste em um aplicativo web simples para gerenciar tarefas pessoais. Os usuários podem adicionar novas tarefas, marcar tarefas como concluídas, editar tarefas existentes e excluir tarefas. Além disso, a aplicação irá buscar dados de duas APIs diferentes para fornecer funcionalidades adicionais, como previsão do tempo e cotação de moedas.

Recursos:

Fetch API:

Utilize uma API para buscar previsão do tempo com base na localização do usuário.
Use outra API para obter informações sobre as cotações de moedas em tempo real.
CRUD de Tarefas:

Implemente um CRUD para gerenciar as tarefas.
Os usuários devem poder adicionar novas tarefas, editar o conteúdo das tarefas, marcar tarefas como concluídas e excluir tarefas.
CRUD de Usuários:

Crie um CRUD básico para gerenciar os usuários.
Os usuários devem poder se cadastrar, fazer login, editar seus dados e excluir suas contas.
Interatividade com o DOM:

Interaja com elementos do DOM para exibir e atualizar a lista de tarefas.
Atualize dinamicamente o conteúdo da página conforme as operações CRUD são realizadas.
Manipulação de Objetos e Arrays:

Manipule objetos para representar cada tarefa, incluindo propriedades como título, descrição e status de conclusão.
Utilize arrays para armazenar e gerenciar a lista de tarefas.
Forms:

Crie um formulário para adicionar novas tarefas.
Permita que os usuários editem as tarefas diretamente na lista, transformando o conteúdo em um campo de input quando clicado.
Criação de Funções:

Organize o código em funções para melhorar a legibilidade e manutenção.
Crie funções para cada operação CRUD e para interagir com as APIs.
